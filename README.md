# Travel With Shubham

Personal travel gallery site — React + Vite + TypeScript + Firebase + Google Drive.

## Admin login
Go to `/login`, sign in with the admin email set up in Firebase Authentication.

## Features
- Public trip gallery (Home page)
- Admin panel to add/delete trips
- Google Drive connect for photo & video uploads (no separate storage cost)
- Per-trip detail page with photo & video gallery
- Drive storage usage display in admin panel
